Loops in Jinja2

For loop and conditional statements

{% for user, os in user_os.items() %} {% if os == "Windows" %}

{{ user }} is a Windows user.

{% elif os == "MacOS" %}

{{ user }} is a Mac user.

{% elif os == "Linux" %}

{{ user }} is a Linux user.

{% else %}

{{ user }} doesn't use a computer.

{% endif %} {% endfor %}
